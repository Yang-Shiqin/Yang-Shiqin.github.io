支持向量机SVM

最初用来解决二分类问题。分割类的超平面离两边的最近的点最远, 判别模型(与概率无关)

三宝: 间隔, 对偶, 核技巧

模型：$f(x)=sign(w^Tx+b)$

  1. hard-margin SVM(最大间隔分类器)
  2. soft-margin SVM
  3. kernel SVM
  4. SVR(SVM做回归)

  2. 求解

推导

几何->数学

  • 样本 ${xi, y_i}{i=1}^N\ ,x_i\in\mathbb R^p, y_i\in{-1, +1}$

hard-margin SVM即最大间隔分类器

  1. 分类器：$f(w)=sign(w^Tx+b)$
  2. 间隔：$margin(w,b)=\min_{x_i}distance(w, b, x_i)$
    • 几何距离，即点到超平面距离：$distance(w, b, x_i)=\frac{y_i}{||w||}(w^Tx_i+b)$
    • 函数距离，函数值，即未归一化的几何距离：$\gamma_i=y_i(w^Tx_i+b)$
    • 几何距离不受wb尺度变化影响，为约束wb(从而得到唯一解)，使min函数距离=1
  3. 最大间隔分类器：

对偶

  1. 拉格朗日乘子法，将 原带约束问题->无约束问题 (约束是对wb而言)
    2. 故无约束原问题
      • 若 $1-y_i(w^Tx_i+b>0$，则 $\max \mathcal L(w,b,\lambda)=+\infty$
      • 若 $1-y_i(w^Tx_i+b\le0$，则 $\max \mathcal L(w,b,\lambda)=\frac{1}{2}ww^T$
  2. 凸优化问题，KKT条件为原对偶问题为强对偶关系的充要条件
    • 弱对偶：$\min\max\mathcal L\ge\max\min\mathcal L$
    • 强对偶：$\min\max\mathcal L=\max\min\mathcal L$
    • KKT：$\begin{cases}\frac{\partial\mathcal L}{\partial w}=\frac{\partial\mathcal L}{\partial b}=\frac{\partial\mathcal L}{\partial\lambda}=0\\\lambda_i(1-y_i(w^Tx_i+b))=0,互补松弛条件\\\lambda_i\ge0\\1-y_i(w^Tx_i+b)\le0\end{cases}$
    • 故对偶问题为
  3. 求解对偶问题等价
    1. $\frac{\partial\mathcal L}{\partial b}=0\rightarrow\sum{i=1}^N\lambda_iy_i=0$ 代入 $\frac{\partial\mathcal L}{\partial w}=0\rightarrow w^*=\sum{i=1}^N\lambda_iy_ix_i$
    2. 代入对偶式子得
    3. $yk(w^Tx_k+b)=1$，$\lambda_k$ 可以不为0，对w有影响，为支持向量，两边同乘 $y_k\rightarrow b^*=y_k-\sum{i=1}^N\lambda_iy_ix_i^Tx_k$

kernel SVM

核方法是一类使用了核函数 $\mathcal K(x^{(i)},x^{(j)})$ 的算法，如，包括 SVM，Gaussian processes，PCA，ridge regression，spectral clustering 等等。使用核技巧可以在原始数据空间下计算其在高维特征空间的相似性，从而避免高维计算内积效率低甚至无法计算，即用 $\mathcal K(x^{(i)},x^{(j)})$ 代替 $\Phi(x^{(i)})^T\Phi(x^{(j)})$

核函数 $\mathcal K(x_i,x_j)$ 参数
线性核 $x_i^Tx_j$
多项式核 $(x_i^Tx_j+1)^d$ d>=1为多项式次数
RBF核 $\exp(-\frac{\ x_i-x_j\ ^2}{\sigma^2})$ $\sigma>0$
高斯核 $\exp(-\frac{\ x_i-x_j\ ^2}{2\sigma^2})$ $\sigma>0$为高斯核的带宽
拉普拉斯核 $\exp(-\frac{\ x_i-x_j\ }{\sigma})$ $\sigma>0$
sigmoid核 $\tanh(\beta x_i^Tx_j+\theta)$ $\beta>0,\theta<0$

映射到高维以可分为:

  • $f(w)=sign (w^{T}\Phi(x_j)+b^)=sign(\sum{i=1}^N\lambda_iy_i\Phi(x_i)^T\Phi(x_j)+y_k-\sum{i=1}^N\lambda_iy_i\Phi(x_i)^T\Phi(x_k))$

故用核技巧可得：

  • $f(w)=sign (w^{T}\Phi(x_j)+b^)=sign(\sum{i=1}^N\lambda_iy_i\mathcal K(x_i,x_j)+y_k-\sum{i=1}^N\lambda_iy_i\mathcal K(x_i,x_k))$

可以不管高维映射，直接定义核函数

  • 核函数必要条件：对称半正定
  • 选择核函数方法：
    • 特征数和样本数差不多，用线性核
    • 特征数少，样本数多，用高斯核
